### 📢 POST EDUCATIVO (série "O que aprendi hoje")

```
💡 O que aprendi hoje sobre Java:

Não importa o quanto você se emprenhe, você sempre será surpreendido com
tantas informações transformadoras

### 🔹 Programação Orientada a Objetos (POO)
Organiza o código em torno de objetos que possuem atributos e métodos.

### 🔹 Classes e Objetos
- **Classe**: modelo que define atributos e comportamentos.
- **Objeto**: instância concreta de uma classe.

### 🔹 Encapsulamento
Protege os dados internos da classe, permitindo acesso controlado via getters e setters.

### 🔹 Herança
Permite que uma classe herde atributos e métodos de outra, promovendo reutilização de código.

### 🔹 Polimorfismo
Capacidade de um objeto assumir diferentes formas:
- **Sobrecarga**: métodos com o mesmo nome e parâmetros diferentes.
- **Sobrescrita**: redefinição de métodos da superclasse.

### 🔹 Abstração
Foca nos aspectos essenciais de um objeto, ocultando detalhes de implementação.

### 🔹 Interfaces
Contratos que definem métodos sem implementação. Classes que implementam devem fornecer os comportamentos.

### 🔹 Modificadores de Acesso
Controlam a visibilidade:
- `public`, `private`, `protected`, *default* (sem modificador).

### 🔹 Pacotes (Packages)
Organizam classes e interfaces em grupos lógicos.

### 🔹 Tratamento de Exceções
Lida com erros em tempo de execução usando `try`, `catch`, `finally`, `throw`.

### 🔹 Coleções (Collections)
Manipulam grupos de objetos: `List`, `Set`, `Map` (ex: `ArrayList`, `HashSet`, `HashMap`).

### 🔹 Generics
Permitem trabalhar com tipos parametrizados, garantindo segurança de tipos.

### 🔹 Threads e Concorrência
Executam tarefas em paralelo, tornando o programa mais eficiente e responsivo.

O mais interessante? Percebi que já aplicava esse conceito sem saber —
quando desenvolvi sistemas de controle de dados lá nos anos 90. 😄

A teoria tem nome bonito. A prática já estava no meu DNA.

Conta aqui: você também já aplicou algum conceito de programação
sem saber que tinha nome técnico?

#Java #POO #BackEnd #ProgramaçãoOrientadaAObjetos #DevBrasil
